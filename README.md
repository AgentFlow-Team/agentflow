# AgentFlow

Landing page and marketing assets for AgentFlow AI automation services.

## Structure

```
website/
├── index.html          # Landing page
├── thank-you.html      # Lead capture thank-you page
├── preview.png         # Open Graph preview image
├── emails/             # Email outreach templates
├── post_images/        # Social media post images
└── products/           # Digital product PDFs and generators
```

## Deploy

`website/index.html` is a self-contained Tailwind CSS landing page. Deploy to any static host (GitHub Pages, Vercel, Netlify).
